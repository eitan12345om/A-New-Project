# A-New-Project
A small project for fun



### Installing requirements
```bash
$ virtualenv venv
$ source venv/bin/activate
$ pip3 install -r requirements.txt
```

### Running locally
```bash
$ python3 server.py
```
Then visit `localhost:5000`
